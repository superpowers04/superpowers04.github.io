Alt Generators, Hosts file repair.

Normally when the sign in button does nothing it's due to a modification to the hosts file that redirects the signin request to a different website (**Normally** caused by mc-leaks which is illegal).
To fix this, remove mc-leaks/Any other account generators( If you have it, If not then just skip this step ), and then do the following:
> **1)** Go to the Start Menu and locate notepad. Right click notepad and select Run as Administrator
> **2)** Press File, then Open. Enter `%SystemRoot%\System32\drivers\etc\hosts` as the "File name" and press "Open"
> **3)** Remove every line mentioning "Mojang" and hit save (Ctrl+S). ( Example: https://imgur.com/a/E8qLgqs )
> You should now be able to play Minecraft again, please never use any illegal tools or accounts again.
> It is highly recommended to sign in and change your password.

Setting up a Minecraft: Java Edition server

Requirements:


  • Java(Oracle Java/OpenJDK 8 is required for 1.13 and below), You can get this from here
  • Minecraft: Java Edition
  • Any Windows or Linux computer, This guide is not garrenteed to work on MacOS

    • Starting out

    Make a new folder anywhere on your system(If you're using Windows then don't use the onedrive folder), Name it anything. This will be our workspace
    Head over to Minecraft's website(https://www.minecraft.net/en-us/download/server) and download the latest version of the Minecraft server jar
    Place the file into your workplace

    Setting up the batch file(Windows)

    Download this and place it in the same folder, then run it, That'll start the server for the first time, You'll see a command prompt window, That's your server console, You can run commands(Like /op or /gamemode) in it and it will tell you what the server is thinking and/or doing

    Setting up pt.2

    Your server should've given you an error mentioning the Eula, Close the window. Go to your workplace folder, You should see a few new files, One of them should be a "eula" or "eula.txt", Double click it and you should get the below or something similar,

    While you probably won't do it, I'd recommend reading the Eula, if you agree then change "eula=false" to "eula=true", After that your server is all setup for playing locally.

    Playing Online(Portforwarding) and Locally

    Playing Locally

    If you are playing with a buddy that is not on the same network as you then skip this and follow the "Port Forwarding" section
    In order for your buddy to join you, You will need to find your IP(Think of it as a passcode or friend code but to access the server)
    Open up command prompt, and in the command prompt type "IPconfig", press enter and look for something like:
    IPv4 Address.......: XXX.XXX.X.X

    For example mine pops up as:
    IPv4 Address......: 192.168.0.106

    This is the IP your buddy will use to connect to the server, Start up the server and then tell your buddy to click "Add a new server" or "Direct Connect"
    For "Add a new server" the name can be anything but the IP needs to be your IPv4 address so tell them to put that there, Click finish and then double click the server and they have connected
    For "Direct connect" tell your buddy to put your IPv4 into the ip box and click connect
    If you want to connect to the server on your own computer then do the above but instead use "localhost" as the IP

    Playing online(Port forwarding)

    If you are playing on the same network with your buddy then please follow the above instead...
    This will be based off of how my router/routers I have used in the past have worked, This is not garrenteed to help you.
    You will need to get the IP address of you and your router,
    Open up command prompt, and in the command prompt type "IPconfig", press enter and look for:
    IPv4 Address.......: XXX.XXX.X.X
    and
    Default Gateway.....:XXX.XXX.XXX.XXX
    For example mine pop up as
    IPv4 Address......: 192.168.0.106 Default Gateway....:192.168.0.1

    The first one is your local IP address
    The second one is your routers IP Address

    Now you are going to want to goto your web browser and type the IP address for your router and you should get a login box.
    You will have to look at the back of your router or search the router online and find the username and password, I know Admin is a popular default username/password.

    Sign in and then find something like "Port Forwarding" or "Virtual servers"
    Mine was under "Advanced", Click "Add", Then put the IP addresss you got from earlier in "IPv4 Address" or "IP address"
    In the "Outbound Port" and "Inbound port" or "Start port" and "end port" put "25565"
    MAKE SURE THAT "Protocol"/"Service Type" IS ETHER "TCP/UDP" OR "BOTH" If you do not have this option then do the step twice, One in TCP and one on UDP

    Add a service name if required, save and test your server, you can get your public ip from Here
    Your buddies will use that to join your server, you will join "localhost", If it works then have fun.
